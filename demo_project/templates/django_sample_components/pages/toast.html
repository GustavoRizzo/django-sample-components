Django Messages (automatic)

The four Django messages queued by the view were displayed as toasts on page load.

Manual — via JavaScript

Call window.showToast(message, type) from anywhere.

HTMX — triggered by server response

Submit the form below. The view sets HX-Trigger: {"showToast": {"message": "...", "type": "success"}} on a valid submission, which fires the toast automatically.

{% async_sum_form %}
Multiple Toasts — via Django Messages + HTMX

The button below fires an HTMX request to the server. The view queues three Django messages and converts them into a single HX-Trigger: {"showToasts": [...]} header using convert_django_messages_to_hx_triggers from django_sample_components.utils.

Usage
{% verbatim %}{% load sample_tags %}

{# 1. Include once — renders container + JS #}
{% simple_toast %}
{% simple_toast position="top-end" %}
{% simple_toast autohide=False delay=3000 %}

{# 2. Django messages fire automatically — no extra code needed #}
# views.py
messages.success(request, "Saved!")

{# 3. Plain JS #}{% endverbatim %}
window.showToast("Hello!", "success")
{% verbatim %}
{# 4. HTMX — single toast from the view: #}{% endverbatim %}
from django_sample_components.utils import get_json_show_toast
response["HX-Trigger"] = json.dumps(get_json_show_toast("Done!", "success"))
{% verbatim %}
{# 5. HTMX — multiple toasts from the view: #}{% endverbatim %}
from django_sample_components.utils import convert_django_messages_to_hx_triggers
messages.success(request, "Saved!")
messages.warning(request, "Check the warnings.")
trigger = convert_django_messages_to_hx_triggers(request)
if trigger:
    response["HX-Trigger"] = json.dumps(trigger)
