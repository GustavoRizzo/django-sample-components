How It Works
  • Clicking the button opens a Bootstrap modal immediately
  • Bootstrap fires show.bs.modal, which triggers an HTMX GET to fetch the registration form
  • The form is re-fetched on every open (always_reload_on_open=True), so the user always starts fresh
  • Field validation (username, subject) and form submission work exactly as in the standalone registration form
