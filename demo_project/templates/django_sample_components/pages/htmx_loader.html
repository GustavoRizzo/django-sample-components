Async HTMX Loader

Generic GET loader that fires an HTMX request on a given trigger and shows a Bootstrap spinner while waiting.

1. Default trigger (load)

Fires immediately on page load. Uses the counter component endpoint.

{% async_htmx_loader url_htmx="/async/counter/component/?initial_value=0&step=1&min_value=0&max_value=10" %}

2. Custom trigger (revealed once)

Fires only when the element scrolls into the viewport.

{% async_htmx_loader url_htmx="/async/counter/component/?initial_value=5&step=2&min_value=0&max_value=20" hx_trigger="revealed once" text_preload="Scroll to reveal counter..." %}

3. Custom loading text

Overrides the default spinner label.

{% async_htmx_loader url_htmx="/async/counter/component/?initial_value=10&step=5&min_value=0&max_value=50" text_preload="Fetching counter from server..." %}

4. Explicit id_target

Provides a stable element ID instead of an auto-generated one.

{% async_htmx_loader url_htmx="/async/counter/component/?initial_value=3&step=1&min_value=0&max_value=10" id_target="my-counter-target" text_preload="Loading my-counter-target..." %}
