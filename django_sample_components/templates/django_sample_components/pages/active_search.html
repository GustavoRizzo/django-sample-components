How It Works
  • Type in the search box — HTMX sends a GET request after 300ms of inactivity
  • Pressing Enter triggers the search immediately
  • On page load, all contacts are shown (empty search returns all)
  • The server filters contacts by first name, last name, or email (case-insensitive)
