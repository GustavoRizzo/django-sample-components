How It Works
  • Click the button — the modal opens immediately with a spinner
  • Bootstrap fires show.bs.modal on the modal element
  • HTMX intercepts that event and sends a GET request to the server
  • The response replaces the spinner with the real content
  • The server is never called unless the user actually opens the popup
