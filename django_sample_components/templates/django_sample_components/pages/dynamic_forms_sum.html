How It Works
  • The form is rendered server-side using django-crispy-forms with the Bootstrap 5 template pack
  • On submit, HTMX POSTs the form data to the component endpoint without reloading the page
  • The server validates the form, computes the result, and returns the updated component HTML
  • HTMX swaps the old component with the new one (hx-swap="outerHTML")
  • Validation errors are displayed inline if the form is invalid
